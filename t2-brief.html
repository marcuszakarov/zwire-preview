How a brief is built

A brief is not a search. It is built.

Four passes stand between a question and a finding you can rely on.

First pass
We source

We gather what is lawfully public and licensed about the subject, within a scope agreed with you — nothing more.

Second pass
We cross-check

No single source is trusted on its own. Each claim has to hold up against independent sources before it earns a place in your brief.

Third pass
We grade

Every finding is given a confidence grade. We never present a guess as a fact, or a fact as a guess.

Last pass
We sign off

A human analyst reads the whole file and signs it before it ever reaches you.

The signature: a grade on every line.

It is the one thing that separates intelligence from rumour — you always know how far to trust each finding.

Confirmed
Independently corroborated.
Probable
Strongly supported, not certain.
Possible
Plausible, limited support.
Unverified
Noted, not yet checked.
